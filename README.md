# AI Agents Landing Page

A visually engaging, modern landing page for promoting AI agent solutions. Built with HTML, CSS (with smooth animations), and minimal JavaScript.

## Features
- Animated gradient header with bold, customer-focused messaging
- Parallax hero section with glassmorphism, floating shapes, and call-to-action
- Use Cases and Benefits sections with interactive cards
- Fully responsive and mobile-friendly
- Minimal, performance-friendly JavaScript (smooth scroll only)

## Usage
1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Customize content, images, and styles as needed for your brand.

## File Structure
- `index.html` — Main landing page
- `style.css` — All styles and animations
- `script.js` — Minimal JS for smooth scroll
- `README.md` — Project overview (this file)

## Customization
- Change colors, images, and text in `index.html` and `style.css` to match your brand.
- Replace the parallax background image URL in `.parallax-section` for a different hero look.
